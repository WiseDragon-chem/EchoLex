# 唤词
***Echolex***
通过算法，在您遗忘的边缘精准唤醒你的记忆。唤词(EchoLex)是一款以个性化与高效率为目标的英语单词学习应用，旨在让每一个单词都记得更科学、更牢固。

# 核心功能
1. 高度定制
轻松创建自己的单词本，支持导入、手动添加等多种方式。
2. 易错诊断
程序会记录并分析你的易错词与易错点，形成你的专属难点词库，并自动加强对这些单词的复习频率和考核力度。
3. 智能唤醒
算法会计算每个单词的最佳复习时间点，并在你即将忘记时自动推送，让记忆事半功倍。

# Echolex  
***Echolex***  
Through algorithms, it precisely awakens your memory on the verge of forgetting. Echolex is an English vocabulary learning app designed for personalization and efficiency, ensuring every word is remembered more scientifically and firmly.  

# Core Features  
1. Highly Customizable  
Easily create your own vocabulary lists, with support for importing, manual adding, and more.  

2. Error Diagnosis  
The program records and analyzes your error-prone words and mistakes, forming a personalized challenging word bank. It automatically increases the review frequency and assessment intensity for these words.  

3. Intelligent Reminders  
The algorithm calculates the optimal review time for each word and automatically sends reminders when you are about to forget, making memory retention twice as effective.
